变量

变量的作用是什么？ 保存数据，使用更方便

使用变量实际上就是使用变量中保存的值

变量的基本使用

命名规则

  1. 开头可以使用英文字母 $ _
  2. js是严格区分大小写的 var a; var A;
  3. 不能使用关键字和保留字

  4. 不能有空格

    建议:变量的命名最好有意义 (困扰这着每一个程序员的问题)

声明

  • var num;//声明一个变量
  • var a, b, c, d;//声明多个变量
  • 怪异的声明方式(不推荐)
  • num = 100;
  • var a = b = c = 100;

赋值 重新赋值

由于js的特性，可以在变量使用的过程中更改变量的类型
我们不推荐这么做，因为会让我们操作出现不经意的问题。
var num;
num = 100;
num = "hehe";
初始化 : 变量在声明的同时给值
var num = 100;

数据类型

5大基本类型

  1. 字符串
    书写字符串必须加双引号或单引号。单引号和双引号必须成对出现

  2. 数值型
    普通数值 0-9 小数 0.1 1.3 极限值（了解）
    特殊值 NaN – not a number

  3. 布尔类型
    true false

  4. undefined
    只有自己一个值
    undefined出现的情况： 变量声明未赋值，值为undefined

  5. null
    空

类型转换

  1. 其他类型转换为字符串类型

    强制转换:num.toString():null和undefined没有toString()方法，使用toString()的方式转换String(num):全能，内部也是调用toString方法
    隐式转换：使用+操作，用任意类型加任意字符串都会连接为新的字符串，一般我们为了不改变数据原有含义，一般加上空字符串“”

  2. 其他类型转换为数值类型

    强制转换：

    1. Number();只可以转换成纯数字的 如果含有非数字部分会返回NaN
    2. parseInt();从左往右取值，一直取到不是数字为止。如果开始就不是数字返回NaN

    3. parsefloat()可以取小数

      隐式转换：

      +-/% -/%都可以正常使用 +的使用需要注意 必须放在前面使用

  3. 转换为bool值

    显示转换：

    Boolean();其他类型转换为bool值的结果

    转换结果为false的是：“” 0 undefined null NaN false

    隐式转换

运算符

算数运算符 +-*/%
一元运算符 操作数只有一个 ++ -- 前置和后置的区别和计算方式
比较运算符 ><>=<>=返回bool值
赋值运算符 = += -= *= /= %=操作的是变量自身的值
相等运算符 ==相等!=不等===全等!==不全等
逻辑运算符
1. 当操作数为bool值时：
    逻辑&&两个操作数都为true，结果为true，任何一个为false，结果为false
    逻辑活||两个操作数都为false，结果为false，任何一个为true，结果为true
    逻辑非 ！取反
2. 当操作数不是bool值时
    先进行隐式转换，然后从左往右看，哪个操作数能决定结果就返回哪个原操作数

运算符的优先级

优先级从高到低
()优先级最高

相等运算符 == != === !==
逻辑运算符 先&& 后||

switch语句

switch 语句：一般用于进行单值判断，但是可以进行范围判断，条件和case值比较时是全等比较
switch(condition){
    case value :
        alert("");
    break必须加，否则执行会执行问题，一直执行后面case中的代码
}
使用switch进行范围判断的时候，将条件设置为true

数组

数字的作用，用于保存多个数据。
数组中的数组我们称为数组的元素。
数组中可以保存多个类型的元素，但是不推荐

数组的声明方式

方式1：
    var arr=new Array(1,2,3,4,5)直接添加元素
方式2：
    常用 数组字面量形式
    var arr=[1,2,3,4,5];

数组的长度

数组的length属性表示数组中元素的个数。

数组的索引

数组中的元素按照索引排列，索引从0开始。
我们可以根据索引获取和修改数组中的元素。

数组的长度和索引的关系

数组中最后一项元素的索引为 arr.length-1
数组中新项的索引，为arr.length

数组的基本操作方法

正向遍历
        var arr = ["a", "b", "c", "d", "e"];
        //当我们操作数组中的所有元素时，索引呈现递增的有个数限制的状态，可以使用for循环，进行索引值的获取
        for (var i = 0; i < arr.length; i++) {
          //循环变量i相当于数组arr中每个元素的索引值
          //console.log(i);
          //将i作为我索引值使用，获取arr中每个元素的值
          console.log(arr[i]);
        }

//这种使用for循环操作数组中所有元素的方式，称为数组遍历。
//反向遍历
for (var i = arr.length - 1; i >= 0; i--) {
  //将i作为我索引值使用，获取arr中每个元素的值
  console.log(arr[i]);

}

反向遍历
    for (var i = arr.length - 1; i >= 0; i--) {
    //将i作为我索引值使用，获取arr中每个元素的值
        console.log(arr[i]); 
    }

复杂数据类型可以一次保存多个值，方便存储使用

函数

简介：函数的作用就是在你需要的时候重复使用代码，函数是在调用时执行的可重复使用的代码块。

函授的声明方式

function 函数名(){
    这里面是你要写的代码块，标志着执行函数时，要执行哪些代码
}
函数内的代码本身是不会执行的。在我们需要执行的时候去执行

调用

函数内的代码只有在调用的时候才会执行，执行在你调用的位置。
写发 函数名();

函数名的使用

当我们单独的使用的了函数名时，代表了这个函数的所有代码（函数体）

函数的参数

有时候函数内的代码有一些参与运算的值是不确定的，我们可以在使用函数的时候去添加这些值，去参与到计算中，我们称为参数。
function fun(n){
    console.log(n);
}
在函数中声明的参数，我们称为形参，这个参数本身是没有值的，这个值是使用者在使用函数的时候传入的，这个参数我们称为实参，实际的参数，
fun(100);

函数内的变量

函数内声明的变量在函数外无法访问

函数的返回值

作用：可以让你设置一个函数内的值，可以被函数外访问
return 变量;
return的第二个作用表示跳出这个函数，后面的代码都不执行了。
所以一个函数只能有一个返回值。

函数的大体概括

function fun(n,m){
    return;
}
fun(100,200);
函数的参数和返回值都是可有可无的，什么时候有看需求，
如果函数没有设置返回值，返回值默认为undefined
undefined出现的情况：变量声明未赋值，访问数组中不存在的元素，一个函数没有设置返回值，默认为undefined

变量作用域

全局作用域就是script标签内，函数外。
在全局作用域中声明的变量为全局变量，全局变量在任何地方都可以访问。变量只能在函数内部被访问

作用域的优先级

当我们在某一个作用域内使用一个变量的时候，会优先搜索当前作用域，如果当前作用域已经有了，就使用这个作用域中的变量值，不会向外面寻找。

js解析的过程

1.预解析
2.解析一行执行一行
3.先去寻找你当前作用域中的所有变量声明 var。将变量声明语句提升到当前作用域顶部。
4.变量的赋值语句保留在原始的位置
5.函数提升，先去寻找所有的函数，然后整体提升到顶端

函数的声明方式

function fun(){};函数声明语句
var fun =function(){};函数表达式
由于函数表达式是将一个函数保存在一个变量中，在预解析阶段，遵循变量提升的规则，导致无法在声明之前进行调用

匿名函数和自调用函数

匿名函数就是没有名字的函数
function(){

}
一般是在函数表达式内使用，不能单独存在
自调用函数
(function(a){
    console.log(a);
})(1);
会在书写的位置上执行一次
回调函数
将一个函数作为其他函数的参数进行传递。
函数的类型
检测方式typeof()函数的类型是function
函数注释的使用
在函数前输入/**点击回车，可以输入一些跟函数参数返回值相关的标注。

this

在方法中使用this表示这个方法所属于的对象。
构造函数
在调用构造函数的时候，调用前加new
构造函数的作用就是创建对象，在构造函数内需要做的事情就是添加属性和方法，
使用this，this就代表创造出来的对象，
构造函数的名字，首字母大写

this的问题

在添加循环事件的时候，不要在循环内使用循环变量。可以替换为this

className的使用

操作标签的类名属性不要用class，因为class是js的关键字，所以使用js提供的className去访问类名属性

取消默认事件

对于一些具有默认事件效果的标签，例如a标签的跳转，我们如果不想执行默认的事件效果，可以在事件的后面加上return false

移入移出事件

鼠标移入onmouseover鼠标悬停在元素上方触发一次代码
鼠标移出onmouseout鼠标离开元素触发一次代码

样式的注意点

使用js设置的样式是应用于标签的行内，所以优先级高。
如果在进行动态的类名添加，会出现优先级不够的情况，可以使用！important
