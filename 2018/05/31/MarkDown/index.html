MarkDown语法说明

Markdown的目标是实现「易读易写」。

Markdown 的语法全由一些符号所组成，这些符号经过精挑细选，其作用一目了然。比如：在文字两旁加上星号，看起来就像强调。

Markdown是一种标记语言，所谓标记语言就是用特定的标识符对一段文字进行标记，从而实现使用者对文字的排版，而在此之前，我们实现文字排版都是采用其他的工具(Word、Pages等)实现，但是不免有些弊端

Markdown针对网页博客、邮件、个人笔记等用途时，都是一种简便、高效率的选择，而且Makedown文件可以通过工具很方便的转成html或者pdf格式，相信如果熟练使用之后，会爱上它。
